# 抽奖转盘系统 v2.0

这是一个简单易用的网页版幸运大转盘抽奖工具，支持自定义奖品数量、名称及中奖概率。

## 功能特点

- **灵活配置**：支持设置 3 到 10 个奖品。
- **自定义概率**：可以为每个奖品单独设置中奖概率（总和需为 100%），实现精准控盘。
- **个性化名称**：支持自定义每个奖品的显示名称。
- **流畅动画**：使用 CSS 和 JS 实现平滑的转盘旋转和减速动画效果。
- **简单易用**：分步向导式操作，无需任何编程基础。

## 使用方法

1. **启动项目**：
   直接在浏览器中打开 `index.html` 文件即可运行。

2. **第 1 步：设置奖品数量**
   - 在下拉菜单中选择本次抽奖包含的奖品数量（最少 3 个，最多 10 个）。
   - 点击“下一步”。

3. **第 2 步：配置奖品**
   - 输入每个奖品的名称。
   - 输入每个奖品的中奖概率（百分比）。
   - 系统会自动校验概率总和是否为 100%。如果不为 100%，将无法进入下一步。
   - 确认无误后点击“生成转盘”。

4. **第 3 步：开始抽奖**
   - 点击中间的“开始抽奖”按钮。
   - 转盘旋转结束后会弹出中奖提示。
   - 点击“重新配置”可以返回第一步重置所有设置。

## 技术栈

- **HTML5**: 页面结构
- **CSS3**: 样式与动画 (Conic Gradients, Transitions)
- **JavaScript (ES6+)**: 逻辑控制与 DOM 操作

## 注意事项

- 请确保使用现代浏览器（Chrome, Firefox, Safari, Edge 等）以获得最佳体验，因为项目使用了 ES6 语法和 CSS3 新特性。

# Lucky Wheel System v2.0

This is a simple and easy-to-use web-based lucky wheel tool that supports customizing the number of prizes, names, and winning probabilities.

## Features

- **Flexible Configuration**: Supports 3 to 10 prizes.
- **Custom Probabilities**: Set individual winning probabilities for each prize (total must be 100%).
- **Personalized Names**: Customize the display name for each prize.
- **Smooth Animation**: Uses CSS and JS for fluid rotation and deceleration effects.
- **User Friendly**: Step-by-step wizard interface.

## How to Use

1. **Start Project**:
   Open `index.html` in your browser.

2. **Step 1: Set Number of Prizes**
   - Select the number of prizes (minimum 3, maximum 10).
   - Click "Next".

3. **Step 2: Configure Prizes**
   - Enter the name for each prize.
   - Enter the winning probability (percentage) for each prize.
   - The system automatically validates that the total is 100%.
   - Click "Generate Wheel".

4. **Step 3: Start Draw**
   - Click the "Start" button in the center.
   - A winner will be announced after the wheel stops.
   - Click "Restart" to reset all settings.

## Tech Stack

- **HTML5**: Structure
- **CSS3**: Styling and Animations (Conic Gradients, Transitions)
- **JavaScript (ES6+)**: Logic and DOM manipulation

## Note

- Please use a modern browser (Chrome, Firefox, Safari, Edge, etc.) for the best experience.
